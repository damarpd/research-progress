# D research

## Research Overview
This study poses four key features: social force model, passenger dynamics, adaptive behavior (responses), mega event/service disruption (both planned and unplanned), by using rail transit system as a case study. This study is based on *social force model* developed by Dirk Helbing and Peter Molnar. This research will explore towards passenger behaviour within the recent phenomena on rail transit system services.
